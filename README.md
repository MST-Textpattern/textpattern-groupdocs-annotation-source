GROUPDOCS Annotation

GroupDocs Annotation plugin for Textpattern

With GroupDocs Annotation plugin for Textpattern you can easily view on website PDF's, Word documents, Excel documents, Powerpoint documents and more with the GroupDocs Annotation tool, directly from within your Textpattern website.

INSTALLATION

	1. To install: textpattern > admin > plugins
	2. Paste the text-code (at .txt file) into the 'Install plugin' box
